ハッカソン クイック招待

Bot ID を貼るだけで、ハッカソン標準権限のついた Discord 招待 URL を生成します。

Discord Developer Portal の Application ID（17〜20 桁の数字）。

❓ Bot ID ってどこで取るの？
  1. Discord Developer Portal を開く
  2. 対象アプリをクリック → 左メニュー「General Information」
  3. 「APPLICATION ID」をコピーしてここに貼り付け
⚠️ 招待前に必ずチェック: Message Content Intent を ON に

Bot がメッセージ本文を読む場合、OAuth2 権限とは別に Gateway Intent を有効化する必要があります（デフォルト OFF）。

  1. Developer Portal → 対象アプリ → 左メニュー「Bot」
  2. 「Privileged Gateway Intents」セクションの MESSAGE CONTENT INTENT を ON

OFF のまま招待すると、mention されていないメッセージの content が空文字で返り「bot がメッセージを読めない」状態になります。

✅ この URL で付与される権限（10 個）
  • View Channel（チャンネル閲覧）
  • Send Messages（メッセージ送信）
  • Create Public Threads（スレッド作成）
  • Send Messages in Threads（スレッド内送信）
  • Embed Links（埋め込み）
  • Attach Files（ファイル添付）
  • Add Reactions（リアクション）
  • Read Message History（履歴読み取り）
  • Use External Emojis（外部絵文字）
  • Use Application Commands（スラッシュコマンド応答）
